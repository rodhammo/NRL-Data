# Visualisations

Interactive charts and data exploration notebooks for NRL data.

## Notebooks

### `round_visulisations.ipynb`
Interactive Plotly charts for team performance analysis:
- Team performance trends across rounds
- Win/loss patterns and points scored vs. conceded
- Home/away performance comparison
- Charts colour-coded with official team colours from `ENVIRONMENT_VARIABLES.py`

## Related
- `player_stats_review.ipynb` (project root) — Season-long player performance tracking
- `scraping/html_interfaces/player.html` — Standalone HTML viewer for player stats (loads from S3)
